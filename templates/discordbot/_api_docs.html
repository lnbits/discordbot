Discord Bot: Connect Discord users to LNbits.

Easily deploy a discord bot to use for your own community! API Docs
Created by jacksn Chris Lennon
Depends on User Manager, by Ben Arc

Registering a discord application

You will first have to create a discord application at https://discord.com/developers/applications, and add a bot to it. You can now connect this extension by opening the Create Bot dialogue and pasting the bot token from discord.

Directly running on instance

The easiest way to get a bot running is to run it directly on your instance by sure the standalone option in the create bot dialogue is disabled

Self-hosting

If you want more control or don't want to add additional load to your instance, you can self-host the bot (standalone box ticked). Details here
To connect to this instance you can use this as your .env
LNBITS_URL={{ request.base_url }}
{% raw %} LNBITS_ADMIN_KEY={{ g.user.wallets[0].adminkey }}
{% endraw %} DATA_FOLDER=./data
